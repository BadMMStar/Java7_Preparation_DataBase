# Java7_Preparation_DataBase
Создали класс State {

  * Объявили 4 переменных с модификатором доступа private.
  
  * Добавили setter'ы и getter'ы для каждой переменной.
  
  * Реализовали проверки для строковых переменных с помощью метода isEmpty(). 
    Если строка у нас пустая это true и выводит сообщение об ошибке. Иначе записывает значение в объект.\
    
  * Реализовали проверки для целочисленных переменных. На отрицательные числа
  
  * В задании присутствуют ограниченный перечень значений. Континент. Мы использовали enum перечисление в Java
    Точку с запятой можно не ставить, если это перечисление без конструкций и методов. Вывели наше значение.
}

  **Замечание:** Обработку исключений я не использовал в моем коде, так как ввод значений у меня типа Unchecked (Runtime exception) = ошибка в работе программы
  Я не могу допустить ввода значений отрицательных чисел площади или числа население. И пустой строки в поле страны, и столицы.
             А вот если бы была у меня запись в файл то тут даже сам IDE предложил бы мне сделать обработку исключений. 
             Потому что это наиболее вероятно что файл может быть переименован или перемещен в другое место.
